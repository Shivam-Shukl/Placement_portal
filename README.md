📌 Placement Portal (Flask Web Application)

The Placement Portal is a web-based application developed using Flask, Python, SQLite, HTML, and CSS to manage campus recruitment activities efficiently. The system provides separate dashboards for Admin, Companies, and Students to streamline the placement process.

🚀 Features

Student Registration & Login

Company Registration & Approval by Admin

Admin Dashboard to manage students, companies, and placement drives

Create and Manage Placement Drives

Students can Apply for Drives

Admin can View, Approve, or Reject Applications

Track Application Status

🛠️ Tech Stack

Backend: Python (Flask)

Database: SQLite with SQLAlchemy

Frontend: HTML, CSS

Version Control: Git & GitHub

🎯 Objective

The goal of this project is to simplify the campus placement process by providing a centralized system where students can apply for jobs, companies can post drives, and administrators can manage all recruitment activities efficiently.
